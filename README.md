# Ashish Kosana Portfolio

Vanilla HTML/CSS/JS portfolio with dark, data-editorial visual style.

## Local setup

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080`.

## Structure

- `index.html` semantic sections, SEO metadata, schema.org Person markup
- `style.css` BEM-ish component styling + responsive breakpoints
- `main.js` typed hero role, project filters, IntersectionObserver reveals, particle canvas
- `assets/` static assets (resume, OG image)
